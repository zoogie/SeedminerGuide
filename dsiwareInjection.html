3DS Guide for Seedminer


Getting your Game patched

Section I - Using the DIS

If the website is down for whatever reason, you can use this one instead

  1. Open this website
  2. Enter your name in the Username field - any username will work, but don't make it too short or include spaces
  3. Select the region you purchased your DSiWare Game in
  4. Upload the game XXXXXXXX.bin you previously saved to your Desktop in the DsiBin field
  5. Upload your movable.sed in the MovableSed field
  6. Click Build and wait a few seconds
  7. Refresh the page every minute to check if a new build is available under Build history on the left and download the file
    • Make sure this build has your name on it. If not, this is not your build
  8. Rename it by removing the .patched_yourname part, so it ends in .bin - for example XXXXXXXX.bin

Section II - Re-Import the Game

  1. Power off your 3DS and insert your SD Card into your PC
  2. Navigate to Nintendo 3DS > Your ID0 > 32 character folder > Nintendo DSiWare
  3. Copy the patched XXXXXXXX.bin file you downloaded and renamed into this folder, overwriting the file that is already there.
  4. Reinsert the SD Card into your 3DS and power it on
  5. Open System Settings, navigate to Data Management > DSiWare
  6. Tap SD Card, tap your game and tap copy. Overwrite the existing copy
    • Your game may appear as Sudoku even if it isn't the purchased game
  7. Exit System Settings and start your game
    • You should be greeted by a white screen complaining that it could not load a file (or solid light-green screens, top and bottom, with no text at all). this is the intended result
    • NOTE: You may need to load further into the game so it loads the modified save file
    If the game boots up normally, you have done something wrong. Redo the process from Section I above

Section III - Preparing your SD Card

  1. Power off your 3DS and insert your SD Card into your PC
  2. Download the latest Version of Luma3DS
  3. Download the latest Version of b9sTool
  4. Download the latest Version of Boot9strap (NOT the devkit, NOT the ntr version)
  5. Download the latest Version of the Homebrew Menu
  6. Download the .firm files corresponding to your system, so either OLD or NEW 2/3DS
    You will need a torrent client like deluge or qbittorent to download the file!
    These files work on firmware version 11.4 / 11.5 / 11.6!
    • 2.54-0_11.4_OLD.firm
    • 2.54-0_11.4_NEW.firm
  7. Copy the boot.firm from the Luma3DS 7z archive to the root of your SD Card
  8. Copy the boot.3dsx (Homebrew Menu) to the root of your SD Card
  9. Copy the boot.nds to the root of your SD Card
  10. Copy the boot9strap.firm from the boot9strap.zip file to the /boot9strap/ folder on your SD Card (Create it if it does not exist)
  11. Copy the 2.54-0_11.4_OLD.firm or 2.54-0_11.4_NEW.firm into the /boot9strap/ folder on your SD Card

Section IV - Installing boot9strap and configuring Luma3DS

  1. Launch b9sTool by starting your DSiWare game
    • You may need to load further into the game so it loads the modified save file
  2. Select "Dump F0F1" to backup the 3DS FIRM
  3. Select "Install boot9strap" and confirm
  4. Wait for it to say "done!". Exit b9sTool and power off your 3DS
    • You may need to force power off by holding the power button
  5. Boot up your 3DS to launch the Luma3DS configuration menu
    • If your 3DS boots normally to the HOME Menu then boot9strap has not installed successfully
      Replace your prep files with freshly downloaded files and restore the F0F1 backup in b9sTool before attempting to install boot9strap again
  6. Use the (A) button and the D-pad to turn on "Show Nand or user string in System Settings"
  7. Press (Start) to save and reboot

You can now continue with the Finalizing Setup
